# Homebrew Tap

Homebrew formulae for [pacificsky](https://github.com/pacificsky) projects.

## Installation

```bash
brew tap pacificsky/tap
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [cage](https://github.com/pacificsky/cage) | Run coding agents safely in containers on macOS |

### cage

```bash
brew install cage
```
